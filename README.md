# SQL College Management System

## Project Overview

This project simulates a real-world College Management Database built using SQL.

The database is designed to manage students, courses, trainers, enrollments, attendance, and marks while supporting analytical reporting and business queries.

---

## Objectives

* Design a relational database from scratch
* Implement primary and foreign key relationships
* Practice SQL joins and aggregations
* Track student attendance and performance
* Generate management reports using SQL

---

## Database Modules

### Students

Stores student information.

### Courses

Stores course details.

### Trainers

Stores trainer information.

### Enrollments

Maps students to courses.

### Attendance

Tracks student attendance records.

### Marks

Stores examination and assessment scores.
